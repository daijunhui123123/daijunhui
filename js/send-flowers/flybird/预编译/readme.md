# 声明提升  
  var 声明的变量会存在声明提升   



  函数整体提升   

  # 预编译发生在函数体内时  
    1. 创建函数的AO对象（Action Object）
    2. 找形参和变量声明，将形参和变量声明作为AO的属性名，赋值与undefined
    3.将形参和实惨统一
    4.在函数体内找函数声明，将函数作为AO对象的属性名，值赋予函数体  
# 预编译发生在全局  
 1. 创建GO对象（Global Object）  
 2. 变量声明，将形参和变量声明作为Go的属性名，赋值与undefined  
 3. 在全局找函数声明，将函数作为GO对象的属性名，值赋予函数体
